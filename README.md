# 药山老师高阶知识管理孵化项目说明

## 目的

1. 和药山老师共同就其价值状态进行梳理、定位并抓紧升级进入市场
2. 利用当下契机，进行资源整合，按照三层知识管理方法进行发展规划
3. 教练能力实践，推促药山老师完成规划，实现产品、变现、研发积累
4. 研发全新的时间管理方法论体系
5. 产出具有药山老师特色的自我管理系统构建案例
6. 呈现抓手思维、目标管理的巨大能量

## 暂定目标

通过构建三层知识管理形态（初级产品化、中级价值变现、高级积累促进）的方式，带动药山老师的价值直接面向市场，并且强化目标达成、时间管理研发。
